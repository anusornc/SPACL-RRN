# Standard Ontology Benchmarks for OWL Reasoner Evaluation

## ORE Competition (OWL Reasoner Evaluation)

The OWL Reasoner Evaluation (ORE) Competition is an annual competition that evaluates OWL 2 compliant reasoners against each other on various standard reasoning tasks. The competition has been running for several years and provides a comprehensive benchmark framework.

### Key Elements from ORE 2015:

1. **Track Structure**: Multiple distinct tracks/divisions/disciplines characterized by problem type (e.g., "effectively propositional" or "OWL 2 EL ontology")

2. **Test Problems**: Derived from a large, neutral, updated yearly set of problems (e.g., for CASC, the TPTP library)

3. **Reasoning Tasks**: Compete primarily on number of problems solved within a tight per-problem timeout

4. **Competition Types**:
   - "Live" competition run during the DL workshop
   - Secondary competition among DL attendees to predict results for various reasoners
   - Custom designed t-shirts for competitors and organizers

### ORE Archive Information:
- Websites for DL2015 and ORE2015 are archived at: http://dl.kr.org/dl2015/

## Standard Benchmark Ontologies

Based on research findings, the following are commonly used standard ontologies for reasoner evaluation:

### 1. ORE Competition Corpus
- Large dataset of ontologies of varying sizes and complexity
- Updated yearly for competition use
- Covers different OWL 2 profiles and expressiveness levels

### 2. LUBM (Lehigh University Benchmark)
- Well-established benchmark for ontology systems
- Focuses on university domain
- Scalable benchmark with configurable size

### 3. OWL 2 EL Profile Ontologies
- Life science ontologies in OWL 2 EL
- Large-scale real-world ontologies
- Tractable reasoning complexity

### 4. Biomedical Ontologies
- Gene Ontology (GO)
- SNOMED CT
- National Cancer Institute Thesaurus (NCIT)
- Human Phenotype Ontology (HPO)

### 5. W3C OWL 2 Test Cases
- Official conformance test cases
- Cover all OWL 2 language features
- Include positive and negative entailment tests
